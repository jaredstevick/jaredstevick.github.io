Hello There!

I'm Jared Stevick and I'm originally from Pittsburgh, PA. For the past ten years, I have worked in finance and customer service but was always looking for something more. While on an extended vacation in the Virgin Islands, I realized that it was time for a career change. I've always been fascinated by technology and have a passion for learning new skills. I also needed something more challenging that provided me with an opportunity for perpetual learning. I found all of those factors and then some in web development.

At the beginning of this year, I moved to Orlando to attend the spring Front-End Engineering cohort at the Iron Yard. While there, I spent twelve crazy weeks learning HTML, CSS, and a whole lot of Javascript. I also became skilled with development processes and tools like Git and Github, Gulp, Grunt, and Bower. Today, I'm building responsive, mobile-friendly websites, doing awesome stuff with API data, and writing Angular applications.

If I'm not at my computer, you can probably find me outside. I love to travel and live for adventure - from backpacking the national parks to diving in the Caribbean, kayaking the lakes and canals of central Florida, mountain biking in the hills of rural Pennsylvania. I'm also not too bad at disc golf. When I'm at home, I enjoy making and building things and growing my own fruits and vegetables in my soilless garden. I'm also a pretty good bartender.

You can check out some of my recent projects below. Feel free to get in touch with me via the links at the bottom of the page!

whatsPup

whatsPup is an Angular web application for pet-sitters to store their client data and send real-time updates when they visit their client's homes. Users are authenticated through Facebook. Client data is stored in Firebase. Email updates are sent via Mandrill and image uploads are made possible through a Cloudinary plugin.

Weather App

This weather app was one of the first projects I worked on when learning how to get and post API data. It uses the Google Maps API to return latitude and longitude when given a zip code. These coordinates are then used to retrieve an accurate local forecast from the Forecast.io API. The app features animated backgrounds and is still a work in progress. The next feature I hope to include is a dynamic background image, based on the current temperature.

  • Live Demo: WeatherApp
  • Date: February 2015
  • Source:

Hex Color Clock

The Hex Color Clock was a quick little project I did when first learning Javascript. The page background color is updated every second. This color is set by converting the current time into a hexadecimal string which corresponds to a particular color.
